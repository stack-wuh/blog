### Bundle 压缩

使用 webpack 的 `compression-webpack-plugin` 插件压缩 Next.js 的 bundle 文件：

```javascript
const CompressionPlugin = require('compression-webpack-plugin')

module.exports = {
  webpack (config) {
    config.plugins.push(new CompressionPlugin())
    return config
  }
}
```

**注意版本问题：** `compression-webpack-plugin` 版本太高在打包编译时会报错 `TypeError: Cannot read property 'tapPromise' of undefined`，降低包版本可解决。

参考：[How to reduce bundle for Next.js](https://blog.devaman.dev/how-to-speed-up-your-next-js-app-using-gzip-ck8o7rr15000axrs1a5fj4nso)

# 技术方案

## Claude.ai 配色参考

**暗色模式为主（claude.ai 默认深色）：**
- 背景：`#0D0D0D` (最深) / `#1A1A1A` (卡片) / `#212121` (悬浮)
- 品牌色/链接：`#D97706` (琥珀) / `#E07B2C`
- 主文字：`#ECECEC` / 次要：`#9CA3AF`
- 代码块背景：`#1E1E1E`
- 边框：`#2A2A2A`

**亮色模式：**
- 背景：`#FAFAFA` / 卡片：`#FFFFFF`
- 品牌色保持一致：`#D97706`
- 主文字：`#1A1A1A` / 次要：`#6B7280`
- 代码块背景：`#F5F5F5`
- 边框：`#E5E5E5`

## 实现方式
通过覆盖 VitePress 全部 `--vp-c-*` CSS 变量，利用 `:root` 和 `:root:not(.dark)` 选择器实现双模式。VitePress 内置亮暗切换，无需额外 JS 逻辑。
